Project for downloading datasets for use in ADCHEM preprocessing.
Interfaces with the available APIs for as many datasets as possible.
The intent is that it will download the most updated version possible, going back to previous version where needed.
However, it should warn when it does so. If the time set is too new, some datasets may not be available,
and it should warn and advise how to handle this. (This requires changes to the specific ADCHEM model)
